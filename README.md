Maia Campus Event Management API

This project provides a complete REST API for managing users, rooms, events, tags, and bookings inside the Maia University Campus.

Main Features   
Users

Create, read, update, and delete users.

Rooms

Manage campus rooms.

Check available and occupied rooms by date or date range.

Tags

Categorize events using tags.

Events

Full CRUD of events.

Time conflict detection.

Search events by keywords.

Filter events by date.

List events by room or by tag.

Bookings

Register users for events.

Prevent duplicate bookings.

List users booked for an event.

Technologies Used

Node.js + Express - API server
MySQL - Database
Docker & Docker Compose - Containerization
OpenAPI 3.0 (Swagger) - API documentation
